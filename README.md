# Voidbear Go Modules

- [cmdargs](./cmdargs/README.md): Structured command-line argument handling.
- [crypto](./crypto/README.md): Encrypting/decrypting data using aes.
- [env](./env/README.md): get, set, rm, expand environment variables. append/prepend path.
- [exec](./exec/README.md): enhances the os/exec module: adds command piping, finding exes, logging.
- [secrets](./secrets/README.md): Generate secrets, mask secrets.
